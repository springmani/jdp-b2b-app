== Frequently Asked Questions ==

= What add-ons do I need to export ACF data? =

To export Advanced Custom Fields (ACF) data, you must have the free ACF Export Add-On active on your site, along with WP All Export. If you’re using the ACF Export Pro Add-On, you must have the free ACF Export Add-On and WP All Export Pro active along with it.

= How do I export ACF data to CSV? =

1. Go to All Export › New Export and select the post type with the ACF Fields.
2. Use our Drag & Drop interface to select the ACF fields that you want to export.
3. Select the Export Type as CSV file.
4. Generate the export file and download it.

= How do I export ACF data to Excel? =

1. Go to All Export › New Export and select the post type with the ACF Fields.
2. Use our Drag & Drop interface to select the ACF fields that you want to export.
3. Select the Export Type as Excel file.
4. Generate the export file and download it.

= How do I export ACF data to XML? =

1. Go to All Export › New Export and select the post type with the ACF Fields.
2. Use our Drag & Drop interface to select the ACF fields that you want to export.
3. Change the Export Type to Feed > Simple XML Feed.
4. Generate the export file and download it.

= How do I export ACF data to Google Sheets? =

1. Go to All Export › New Export and select the post type with the ACF Fields.
2. Use our Drag & Drop interface to select your ACF export columns.
3. Run the export.
4. Connect to Google Sheets using Zapier.

= How do I migrate ACF data to another website? =

1. Export the ACF data that you want to migrate using WP All Export.
2. Download the Bundle file, which contains import instructions.
3. On the destination site, upload the bundle file using WP All Import.
4. Finish the import to complete the migration.
5. Verify that the ACF data was properly migrated.

= How do I bulk edit ACF data? =

1. Export the ACF fields data that you want to edit.
2. Use a spreadsheet app to modify the contents of the export file.
3. When you've completed your edits, use the Import with WP All Import option to import the file back into WordPress.
4. Once the import is finished, verify that your edits were correctly applied.

= How to schedule your ACF export? =

You can schedule your exports of ACF fields data directly through our interface by using our Automatic Scheduling Service. Alternatively, you can create cron jobs manually on your server. Either approach will work, though our scheduling service is definitely more convenient.
